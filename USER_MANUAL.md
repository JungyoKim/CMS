## 1. 시스템 개요

### 1.1 시스템 소개

**고객 관리 시스템(CMS)**은 계약 기반의 고객, 제품, 펌웨어, AS 기록을 통합 관리하는 기업용 웹 애플리케이션입니다. 계약서 관리, 고객 정보 관리, 제품 및 펌웨어 관리, AS 이력 추적 등의 기능을 제공합니다.

### 1.2 주요 기능

- **계약 관리**: 계약서 등록, 수정, 조회, 첨부파일 관리
- **고객사 관리**: 고객사 정보 등록 및 관리, 담당자 정보 관리
- **제품 관리**: 제품 정보 등록, 관리코드 및 가격 정보 관리
- **펌웨어 관리**: 프로토콜/펌웨어 버전 관리, 파일 업로드
- **AS 관리**: 고객 지원 기록 관리, 진행 상태 추적
- **대시보드**: 사전영업, AS 진행 상황 한눈에 확인

### 1.3 시스템 요구사항

- **지원 브라우저**: Chrome, Edge, Firefox, Safari (최신 버전 권장)
- **인터넷 연결**: 안정적인 인터넷 연결 필요
- **화면 해상도**: 최소 1280x720, 권장 1920x1080 이상

## 2. 시작하기

### 2.1 로그인

1. 시스템 URL로 접속합니다.
2. 로그인 페이지에서 비밀번호를 입력합니다.
3. 비밀번호를 입력한 후 로그인 버튼을 클릭합니다.

> **참고**: 비밀번호는 시스템 관리자에게 문의하세요. 비밀번호 변경은 설정 메뉴에서 가능합니다.

### 2.2 화면 구성

로그인 후 화면은 다음과 같이 구성됩니다:

- **좌측 사이드바**: 메뉴 네비게이션
  - 홈
  - 계약 관리
  - 고객사 관리
  - 제품 관리
  - 펌웨어 관리
- **상단 헤더**: 현재 페이지 제목, 다크모드 토글
- **메인 콘텐츠 영역**: 각 메뉴의 데이터 테이블 및 기능

### 2.3 기본 조작법

- **사이드바 접기/펼치기**: 좌측 상단 메뉴 아이콘 클릭
- **다크모드 전환**: 상단 헤더의 태양/달 아이콘 클릭
- **검색**: 각 페이지 상단의 검색창 사용
- **데이터 추가**: 테이블 상단의 "추가" 버튼 클릭
- **데이터 수정/삭제**: 각 행의 작업 메뉴(⋮) 아이콘 클릭

## 3. 홈 대시보드

홈 화면에서는 다음 세 가지 주요 정보를 한눈에 확인할 수 있습니다.

### 3.1 사전영업중 고객

계약 상태가 "사전영업"인 고객 목록을 계약일 오래된 순으로 최대 10개 표시합니다.

**표시 정보**:
- 고객사명
- 발주처
- 계약명
- 총 금액 (계약금 + 선금 + 중도금 + 잔금)
- 주소

**기능**:
- 행 클릭 시 해당 계약 상세 정보 확인 및 수정 가능

### 3.2 AS미완료

완료되지 않은 AS 기록을 요청일 오래된 순으로 최대 10개 표시합니다.

**표시 정보**:
- 고객사명
- 요청일자
- 계약명
- 요청내용
- 비용

**기능**:
- 행 클릭 시 해당 계약 상세 정보 확인 및 AS 기록 관리 가능

### 3.3 최근 AS완료 고객

최근 완료된 AS 기록을 완료일 최신 순으로 최대 10개 표시합니다.

**표시 정보**:
- 고객사명
- 요청일자
- 계약명
- 요청내용
- 대응일자
- 대응내용
- 비용

**기능**:
- 행 클릭 시 해당 계약 상세 정보 확인 가능

## 4. 계약 관리

계약 정보를 등록, 조회, 수정, 삭제할 수 있는 메인 기능입니다.

### 4.1 계약 목록 조회

**접근 방법**: 좌측 사이드바에서 "계약 관리" 메뉴 클릭

**표시 정보**:
- 계약 ID
- 계약명
- 고객사명
- 발주처
- 계약 상태 (사전영업/진행중/해지)
- 계약일
- 총 금액
- AS 상태 (없음/진행중/완료)
- 첨부파일

### 4.2 검색 기능

계약 목록 상단의 검색창에서 다양한 조건으로 검색할 수 있습니다.

**검색 필드**:
- **계약명**: 계약서 이름으로 검색
- **고객사명**: 고객사 이름으로 검색
- **이메일**: 고객사 담당자 이메일로 검색
- **주소**: 고객사 주소로 검색
- **고객상태**: 계약 상태로 검색 (사전영업/진행중/해지)
- **첨부파일명**: 첨부된 파일 이름으로 검색
- **AS상태**: AS 진행 상태로 검색 (없음/진행중/완료)

**검색 방법**:
1. 검색 필드 드롭다운에서 검색 조건 선택
2. 검색어 입력
3. Enter 키 입력 또는 검색 아이콘 클릭

### 4.3 계약 추가

1. 계약 관리 페이지 상단의 **"추가"** 버튼 클릭
2. 계약 정보 입력 폼에 다음 정보를 입력합니다:

#### 기본 정보
- **계약명** (필수): 계약서 이름
- **고객사**: 드롭다운에서 고객사 선택 (등록되지 않은 경우 고객사 관리에서 먼저 등록)
- **발주처**: 드롭다운에서 발주처 선택 (선택사항)
- **계약 상태**: 사전영업 / 진행중 / 해지 중 선택
- **계약일**: 달력에서 선택
- **계약해지일**: 계약 해지 시 선택 (선택사항)
- **영업시작일**: 영업 시작일 선택 (선택사항)

#### 금액 정보
- **계약금**: 숫자 입력
- **선금**: 숫자 입력
- **중도금**: 숫자 입력
- **잔금**: 숫자 입력
- **계좌번호**: 입금 계좌번호 (선택사항)
- **세금계산서 발행일**: 달력에서 선택 (선택사항)

#### 유지보수 정보
- **유지비 월 청구금액**: 숫자 입력 (선택사항)
- **청구일 매월 (일자)**: 1~31 사이 숫자 입력 (선택사항)

#### 담당자 정보
- **계약담당자 이름**: 텍스트 입력
- **계약담당자 전화번호**: 텍스트 입력
- **계약담당자 이메일**: 이메일 형식 입력

#### 구축 및 설치 정보
- **구축기간 시작일**: 달력에서 선택
- **구축기간 종료일**: 달력에서 선택
- **설치사**: 텍스트 입력
- **설치담당자 이름**: 텍스트 입력
- **설치담당자 전화번호**: 텍스트 입력

#### 기타 정보
- **건물정보**: 텍스트 입력 (선택사항)
- **기타 메모**: 텍스트 입력 (선택사항)

#### 첨부파일
- **첨부파일 업로드**: 파일 선택 버튼을 클릭하여 계약서 관련 문서 업로드
  - 지원 형식: PDF, 이미지 파일 등
  - 여러 파일 업로드 가능

3. **"저장"** 버튼 클릭

### 4.4 계약 상세 정보 관리

계약 목록에서 각 행의 오른쪽 **작업 메뉴(⋮)** 아이콘을 클릭하면 다음 작업을 수행할 수 있습니다:

- **상세보기/수정**: 계약 정보 확인 및 수정
- **삭제**: 계약 삭제 (소프트 삭제)

### 4.5 계약 상세 정보

계약 상세보기/수정 다이얼로그에서는 다음 추가 기능을 사용할 수 있습니다:

#### 객실 관리
- **객실 추가**: 객실 정보 추가
  - 건물명
  - 룸번호
  - ID (룸 코드)
  - 메모
- **객실 목록**: 등록된 객실 목록 표시
- **객실 수정/삭제**: 각 객실의 작업 메뉴에서 수정 또는 삭제

#### 중계기 관리
- **중계기 추가**: 중계기 정보 추가
  - 중계기 ID
  - 룸번호목록 메모
  - 메모
- **중계기 목록**: 등록된 중계기 목록 표시
- **중계기 수정/삭제**: 각 중계기의 작업 메뉴에서 수정 또는 삭제

#### 설치제품 관리
- **설치제품 추가**: 설치된 제품 정보 추가
  - 제품: 드롭다운에서 선택
  - 프로토콜(펌웨어): 드롭다운에서 선택 (선택사항)
  - 수량: 숫자 입력
  - 메모: 텍스트 입력 (선택사항)
- **설치제품 목록**: 등록된 설치제품 목록 표시
- **설치제품 수정/삭제**: 각 설치제품의 작업 메뉴에서 수정 또는 삭제

#### AS 기록 관리
- **AS 기록 추가**: 고객 지원 기록 추가
  - 요청일자: 달력에서 선택
  - 요청내용: 텍스트 입력
  - 대응일자: 달력에서 선택 (선택사항)
  - 대응내용: 텍스트 입력 (선택사항)
  - 비용: 숫자 입력 (선택사항)
  - 완료여부: 체크박스로 표시
- **AS 기록 목록**: 등록된 AS 기록 목록 표시
  - 완료되지 않은 AS는 빨간색으로 강조 표시
- **AS 기록 수정/삭제**: 각 AS 기록의 작업 메뉴에서 수정 또는 삭제

#### 관련문서 관리
- **문서 업로드**: 계약과 관련된 추가 문서 업로드
- **문서 목록**: 업로드된 문서 목록 표시
  - 문서 제목
  - 원본 파일명
  - 파일 크기
  - 업로드 일시
- **문서 다운로드**: 각 문서의 다운로드 아이콘 클릭
- **문서 삭제**: 각 문서의 삭제 아이콘 클릭

### 4.6 계약 삭제

1. 계약 목록에서 삭제할 계약의 **작업 메뉴(⋮)** 클릭
2. **"삭제"** 선택
3. 확인 다이얼로그에서 **"삭제"** 확인

> **주의**: 삭제된 계약은 소프트 삭제되며, 목록에서 숨겨집니다. 데이터는 데이터베이스에 보관됩니다.

### 4.7 페이지네이션

계약 목록 하단의 페이지네이션 컨트롤을 사용하여 다음 작업을 수행할 수 있습니다:

- **페이지 크기 변경**: 페이지당 표시할 항목 수 선택 (10, 20, 50, 100)
- **페이지 이동**: 이전/다음 페이지 버튼 또는 페이지 번호 클릭
- **첫 페이지/마지막 페이지 이동**: 양쪽 끝 화살표 버튼 클릭

## 5. 고객사 관리

고객사 정보를 등록, 조회, 수정, 삭제할 수 있는 기능입니다.

### 5.1 고객사 목록 조회

**접근 방법**: 좌측 사이드바에서 "고객사 관리" 메뉴 클릭

**표시 정보**:
- 고객사 ID
- 고객사명 (name1~5 조합)
- 사업자등록번호
- 주소
- 주담당자 이름
- 주담당자 전화번호
- 주담당자 이메일
- 등록증 파일

### 5.2 검색 기능

고객사 목록 상단의 검색창에서 다음 조건으로 검색할 수 있습니다:

- **고객사명**: 고객사 이름으로 검색
- **사업자등록번호**: 사업자등록번호로 검색
- **주소**: 주소로 검색
- **이메일**: 담당자 이메일로 검색
- **전화번호**: 담당자 전화번호로 검색
- **등록증 파일명**: 등록증 파일 이름으로 검색

### 5.3 고객사 추가

1. 고객사 관리 페이지 상단의 **"추가"** 버튼 클릭
2. 고객사 정보 입력 폼에 다음 정보를 입력합니다:

#### 기본 정보
- **고객사명 1~5**: 고객사 이름 구성요소 (예: "주식회사", "ABC", "테크", "코리아", "서울")
- **사업자등록번호**: 숫자 및 하이픈 입력
- **우편번호**: 우편번호 입력 (선택사항)
- **주소**: 주소 입력 (선택사항)
- **팩스**: 팩스번호 입력 (선택사항)

#### 주담당자 정보
- **이름**: 텍스트 입력
- **직책**: 텍스트 입력 (선택사항)
- **전화번호**: 전화번호 입력
- **이메일**: 이메일 형식 입력 (선택사항)

#### 부담당자 정보 (선택사항)
- **이름**: 텍스트 입력
- **직책**: 텍스트 입력
- **전화번호**: 전화번호 입력
- **이메일**: 이메일 형식 입력

#### 등록증 파일
- **등록증 업로드**: 파일 선택 버튼을 클릭하여 사업자등록증 등 문서 업로드
  - 지원 형식: PDF, 이미지 파일 등

3. **"저장"** 버튼 클릭

### 5.4 고객사 수정

1. 고객사 목록에서 수정할 고객사의 **작업 메뉴(⋮)** 클릭
2. **"수정"** 선택
3. 수정 다이얼로그에서 정보 수정
4. **"저장"** 버튼 클릭

### 5.5 고객사 삭제

1. 고객사 목록에서 삭제할 고객사의 **작업 메뉴(⋮)** 클릭
2. **"삭제"** 선택
3. 확인 다이얼로그에서 **"삭제"** 확인

> **주의**: 
> - 삭제된 고객사는 소프트 삭제되며, 목록에서 숨겨집니다.
> - 계약에 연결된 고객사는 삭제 전 해당 계약을 먼저 처리해야 합니다.

## 6. 제품 관리

제품 정보를 등록, 조회, 수정, 삭제할 수 있는 기능입니다.

### 6.1 제품 목록 조회

**접근 방법**: 좌측 사이드바에서 "제품 관리" 메뉴 클릭

**표시 정보**:
- 제품 ID
- 제품명
- 관리코드
- 단가
- 버전
- 프로토콜(펌웨어)
- 제품 사진

### 6.2 검색 기능

제품 목록 상단의 검색창에서 다음 조건으로 검색할 수 있습니다:

- **제품명**: 제품 이름으로 검색
- **관리코드**: 관리코드로 검색
- **프로토콜**: 프로토콜 이름으로 검색
- **제품 사진 파일명**: 제품 사진 파일 이름으로 검색

### 6.3 제품 추가

1. 제품 관리 페이지 상단의 **"추가"** 버튼 클릭
2. 제품 정보 입력 폼에 다음 정보를 입력합니다:

#### 기본 정보
- **제품명** (필수): 제품 이름
- **관리코드**: 제품 관리용 코드 (선택사항)
- **단가**: 숫자 입력 (선택사항, 기본값: 0)
- **버전**: 제품 버전 정보 (선택사항)
- **프로토콜(펌웨어)**: 드롭다운에서 선택 (선택사항, 펌웨어 관리에서 먼저 등록 필요)
- **메모**: 제품 관련 메모 (선택사항)

#### 제품 사진
- **제품 사진 업로드**: 파일 선택 버튼을 클릭하여 제품 사진 업로드
  - 지원 형식: 이미지 파일 (JPG, PNG 등)

3. **"저장"** 버튼 클릭

### 6.4 제품 수정

1. 제품 목록에서 수정할 제품의 **작업 메뉴(⋮)** 클릭
2. **"수정"** 선택
3. 수정 다이얼로그에서 정보 수정
4. **"저장"** 버튼 클릭

### 6.5 제품 삭제

1. 제품 목록에서 삭제할 제품의 **작업 메뉴(⋮)** 클릭
2. **"삭제"** 선택
3. 확인 다이얼로그에서 **"삭제"** 확인

> **주의**: 삭제된 제품은 소프트 삭제되며, 목록에서 숨겨집니다. 계약에 설치된 제품과 연결된 경우에도 삭제가 가능하지만, 데이터 무결성을 위해 주의가 필요합니다.

## 7. 펌웨어 관리

펌웨어(프로토콜) 정보를 등록, 조회, 수정, 삭제할 수 있는 기능입니다.

### 7.1 펌웨어 목록 조회

**접근 방법**: 좌측 사이드바에서 "펌웨어 관리" 메뉴 클릭

**표시 정보**:
- 프로토콜 ID
- 이름
- 버전
- 펌웨어 파일
- 기타문서 파일

### 7.2 검색 기능

펌웨어 목록 상단의 검색창에서 다음 조건으로 검색할 수 있습니다:

- **이름**: 펌웨어 이름으로 검색
- **버전**: 버전 정보로 검색
- **펌웨어 파일명**: 펌웨어 파일 이름으로 검색
- **기타문서 파일명**: 기타문서 파일 이름으로 검색

### 7.3 펌웨어 추가

1. 펌웨어 관리 페이지 상단의 **"추가"** 버튼 클릭
2. 펌웨어 정보 입력 폼에 다음 정보를 입력합니다:

#### 기본 정보
- **이름** (필수): 펌웨어/프로토콜 이름
- **버전**: 버전 정보 (선택사항)
- **메모**: 펌웨어 관련 메모 (선택사항)

#### 파일 관리
- **펌웨어 파일 업로드**: 펌웨어 바이너리 파일 업로드
  - 지원 형식: 바이너리 파일, PDF 등
- **기타문서 파일 업로드**: 펌웨어 관련 추가 문서 업로드 (선택사항)
  - 지원 형식: PDF, 문서 파일 등
  - 여러 파일 업로드 가능

3. **"저장"** 버튼 클릭

### 7.4 펌웨어 수정

1. 펌웨어 목록에서 수정할 펌웨어의 **작업 메뉴(⋮)** 클릭
2. **"수정"** 선택
3. 수정 다이얼로그에서 정보 수정
4. **"저장"** 버튼 클릭

### 7.5 펌웨어 파일 관리

펌웨어 수정 다이얼로그에서 파일을 추가, 다운로드, 삭제할 수 있습니다:

- **파일 추가**: 파일 선택 버튼을 클릭하여 새 파일 업로드
- **파일 다운로드**: 각 파일의 다운로드 아이콘 클릭
- **파일 삭제**: 각 파일의 삭제 아이콘 클릭

### 7.6 펌웨어 삭제

1. 펌웨어 목록에서 삭제할 펌웨어의 **작업 메뉴(⋮)** 클릭
2. **"삭제"** 선택
3. 확인 다이얼로그에서 **"삭제"** 확인

> **주의**: 삭제된 펌웨어는 소프트 삭제되며, 목록에서 숨겨집니다. 제품이나 계약에 연결된 펌웨어인 경우에도 삭제가 가능하지만, 데이터 무결성을 위해 주의가 필요합니다.

## 8. 파일 관리

시스템에서 업로드된 파일들을 관리하는 방법입니다.

### 8.1 파일 업로드

다음 위치에서 파일을 업로드할 수 있습니다:

- **계약 관리**: 계약서 첨부파일, 관련문서
- **고객사 관리**: 사업자등록증 등
- **제품 관리**: 제품 사진
- **펌웨어 관리**: 펌웨어 파일, 기타문서

**업로드 방법**:
1. 각 항목의 파일 선택 영역에서 **"파일 선택"** 버튼 클릭
2. 파일 탐색기에서 원하는 파일 선택 (Ctrl 키를 누른 채로 여러 파일 선택 가능)
3. 선택된 파일이 목록에 표시됨
4. 저장 시 파일이 서버에 업로드됨

### 8.2 파일 다운로드

- 각 파일 목록에서 다운로드 아이콘(⬇) 클릭
- 브라우저의 기본 다운로드 동작으로 파일이 저장됨

### 8.3 파일 삭제

- 각 파일 목록에서 삭제 아이콘(🗑) 클릭
- 확인 후 파일이 삭제됨

### 8.4 파일 형식 제한

시스템은 대부분의 파일 형식을 지원하지만, 다음을 권장합니다:

- **문서**: PDF, DOC, DOCX
- **이미지**: JPG, PNG, GIF
- **펌웨어**: BIN, HEX, 그 외 바이너리 파일

### 8.5 파일 크기 제한

서버 설정에 따라 파일 크기 제한이 있을 수 있습니다. 대용량 파일 업로드 시 시간이 소요될 수 있습니다.

## 9. 시스템 설정

시스템 관리 및 설정을 변경할 수 있는 기능입니다.

### 9.1 설정 메뉴 접근

1. 우측 상단 사용자 메뉴 클릭
2. "설정" 선택

### 9.2 비밀번호 변경

1. 설정 페이지에서 "비밀번호 변경" 섹션으로 이동
2. 다음 정보를 입력:
   - **현재 비밀번호**: 현재 사용 중인 비밀번호
   - **새 비밀번호**: 변경할 비밀번호
   - **비밀번호 확인**: 새 비밀번호 재입력
3. **"비밀번호 변경"** 버튼 클릭

> **주의**: 
> - 새 비밀번호와 확인 비밀번호가 일치해야 합니다.
> - 현재 비밀번호와 새 비밀번호가 같을 수 없습니다.
> - 비밀번호 변경 후 로그인이 유지되며, 다음 로그인부터 새 비밀번호를 사용합니다.

### 9.3 세션 만료 시간 설정

1. 설정 페이지에서 "세션 만료 시간 설정" 섹션으로 이동
2. 다음 옵션 중 선택:
   - **만료 없음**: 세션이 만료되지 않음
   - **분**: 분 단위로 만료 시간 설정 (예: 30분)
   - **시간**: 시간 단위로 만료 시간 설정 (예: 2시간)
   - **일**: 일 단위로 만료 시간 설정 (최대 7일)
3. 값을 입력하고 **"설정 저장"** 버튼 클릭

> **참고**: 세션 만료 시간을 설정하면 해당 시간이 경과한 후 자동으로 로그아웃됩니다. 보안을 위해 적절한 만료 시간을 설정하는 것을 권장합니다.

## 10. 부록: FAQ 및 문제 해결

### 10.1 자주 묻는 질문 (FAQ)

**Q1. 계약을 삭제했는데 다시 복구할 수 있나요?**
A: 시스템은 소프트 삭제 방식을 사용하므로, 삭제된 데이터는 데이터베이스에 보관됩니다. 복구가 필요한 경우 시스템 관리자에게 문의하세요.

**Q2. 파일 업로드가 실패하는 경우는?**
A: 다음을 확인하세요:
- 파일 크기가 서버 제한을 초과하지 않는지
- 인터넷 연결이 안정적인지
- 브라우저 캐시를 지우고 다시 시도

**Q3. 검색 결과가 나오지 않는 경우는?**
A: 다음을 확인하세요:
- 검색 필드가 올바르게 선택되었는지
- 대소문자 구분 없이 검색되므로 정확한 키워드 입력
- 다른 검색 필드로 시도

**Q4. 여러 계약에 동일한 고객사를 연결할 수 있나요?**
A: 네, 하나의 고객사 정보를 여러 계약에서 사용할 수 있습니다.

**Q5. 계약 상태를 변경하는 방법은?**
A: 계약 상세보기/수정 다이얼로그에서 "계약 상태" 필드를 변경한 후 저장하면 됩니다.

### 10.2 문제 해결

#### 문제: 로그인이 안 됩니다
- 비밀번호가 올바른지 확인
- 대소문자 구분 확인
- 브라우저 쿠키가 활성화되어 있는지 확인
- 시스템 관리자에게 비밀번호 재설정 요청

#### 문제: 데이터가 저장되지 않습니다
- 필수 항목이 모두 입력되었는지 확인
- 인터넷 연결 상태 확인
- 브라우저를 새로고침하고 다시 시도
- 오류 메시지가 표시되는 경우 해당 내용 확인

#### 문제: 파일이 업로드되지 않습니다
- 파일 크기 제한 확인
- 파일 형식이 지원되는지 확인
- 브라우저 개발자 도구(F12)의 네트워크 탭에서 오류 확인
- 다른 파일로 시도

#### 문제: 페이지가 느리게 로드됩니다
- 인터넷 연결 속도 확인
- 브라우저 캐시 삭제
- 다른 브라우저에서 시도
- 데이터가 많은 경우 검색 조건을 사용하여 필터링

### 10.3 시스템 제한사항

- **페이지당 표시 항목 수**: 기본 10개, 최대 100개
- **파일 업로드 크기**: 서버 설정에 따라 다름 (일반적으로 수십 MB)
- **동시 접속자 수**: 서버 성능에 따라 다름
- **검색 결과 수**: 성능을 위해 제한될 수 있음

### 10.4 문의 및 지원

기술적 문제나 기능 문의사항이 있는 경우:
- 시스템 관리자에게 문의
- 오류 메시지가 표시되는 경우 해당 내용을 기록하여 전달
- 문제 발생 시점과 작업 내용을 상세히 기록
